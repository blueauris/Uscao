A Rectangular Barn

矩形牛棚

Mircea Pasoi -- 2003

译 by yangzhe1990

到底是个资本家，Farmer John想通过买更多的奶牛来扩大它的生意。它需要给奶牛建造一个新的牛棚。
FJ买了一个矩形的R(1 <= R <= 3000)行C(1 <= C <= 3000)列的牧场。不幸的是，他发现某些1 x 1的区域被损坏了，所以它不可能在把整个牧场建造成牛棚了。
FJ数了一下，发现有P(1 <= p <= 30000)个1 x 1的损坏区域并且请你帮助他找到不包含损坏区域的面积最大的牛棚。

PROGRAM NAME: rectbarn

INPUT FORMAT
?  第1行: 三个空格隔开的整数 R, C, and P.
?  第2..P+1行: 每行包含两个空格隔开的整数, r和c, 给出一个损坏区域的行号和列号.

SAMPLE INPUT (file rectbarn.in)
3 4 2
1 3
2 1

OUTPUT FORMAT
?  第1行: 牛棚的最大可能面积

SAMPLE OUTPUT (file rectbarn.out)
6

OUTPUT DETAILS

  1 2 3 4
.+-+-+-+-+
1| | |X| |
.+-+-+-+-+
2|X|#|#|#|
.+-+-+-+-+
3| |#|#|#|
.+-+-+-+-+
标'X'的区域是损坏的, 标 '#'的区域是牛棚.
